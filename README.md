# music_player
A simple music player built with JavaScript, designed for those moments when you just want to play your music without distractions. It features a clean and minimal interface with essential controls like play, pause, and track navigation.

With API integration currently disabled, the player works entirely with local files, focusing on reliability and simplicity. This project highlights core JavaScript concepts such as DOM manipulation, event handling, and audio control, delivering a straightforward yet functional listening experience.
